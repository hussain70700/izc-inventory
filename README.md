## 🚀 Enterprise Retail Management System
A high-performance Flutter application built for real-time inventory control, sales automation, and staff management. This project features a robust POS system integrated with Firebase Realtime Database and FCM for instant updates and notifications.

## ✨ Key Features
📊 All-in-One Dashboard
Real-time Metrics: Provides a high-level overview of total sales, financial health, and shipment performance.

Performance Tracking: Monitors delivery percentages and return ratios using visual progress indicators.

## 🛒 POS & Sales Automation
Receipt Generation: Create professional sales entries and generate instant digital receipts.

WhatsApp Integration: Automatically sends receipts and transaction details to customers via WhatsApp.

Returns & Exchanges: Dedicated workflow to process item returns or exchanges, ensuring inventory accuracy.

Promo Code Integration: Apply custom-generated promo codes during checkout to provide instant discounts.

## 📦 Intelligent Inventory Tracking
Detailed Audit Logs: Tracks every stock movement with specific reasons (e.g., "Stock increased: Added by Admin").

Live Alerts: Features real-time "Low Stock" and "Out of Stock" indicators to prevent missed sales.

## 👥 Staff & Role Management
Dual-Role System: Supports two distinct roles with tailored permissions:

Manager: Full access to Inventory, Sales, and Staff management.

Sales Person: Restricted access limited strictly to the Sales tab for processing transactions.

Activity Monitoring: Tracks last login times and staff identification for accountability.

## 👤 Customer Insights
Dedicated Customer Tab: Monitor individual customer profiles and purchase history.

Sales Correlation: View every transaction linked to a specific customer to track loyalty and spending habits.

## 🛠️ Tech Stack
Frontend: Flutter (Optimized for Web/Desktop)

Database: Firebase Realtime Database for live stock and sales syncing.
